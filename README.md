The Daily Pour - Coffee Shop Management System
-----------------------------------------------
A simple Python project for managing coffee shop orders.
This project was created as part of a university course assignment.

---

Features:
---------
1. **Predefined Menu**: Select from a list of drinks with fixed prices (e.g., oolong tea, espresso, cappuccino).
2. **Add Orders**: Create orders for customers by selecting items from the menu.
3. **List Orders**: View all current orders with customer names, items, and totals.
4. **Add Items to Order**: Add more items to an existing order by selecting its number.
5. **Remove Order**: Remove an order by selecting its number.
6. **Calculate Total Revenue**: See the total revenue from all orders.

---

How to Run:
-----------
1. Save the files (`models.py`, `main.py`, `requirements.txt`, `readme.txt`) in a folder.
2. Open a terminal or command prompt in the folder.
3. Run the program with the command:
   python main.py

---

Requirements:
-------------
- Python 3.x (no external libraries required)
